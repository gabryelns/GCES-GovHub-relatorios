**<h2>Stack Tecnológica</h2>**

Para entender o funcionamento dos pipelines e dashboards:

* [**Apache Airflow**](https://airflow.apache.org/docs/) - Orquestração de fluxos.
* [**dbt**](https://docs.getdbt.com/) - Transformação de dados.
* [**Apache Superset**](https://superset.apache.org/docs/intro) - Visualização de dados.
* [**PostgreSQL**](https://www.postgresql.org/docs/) - Banco de dados.
* [**Docker**](https://docs.docker.com/) - Gerenciamento de containers.
