# Contribuições GCES ao Gov Hub BR

Bem-vindos! Este portal centraliza e documenta todas as contribuições realizadas pela nossa equipe ao projeto open source **Gov Hub BR**. O trabalho faz parte das atividades práticas da disciplina de **Gerência de Configuração e Evolução de Software (GCES)** da **Universidade de Brasília (UnB)**.

---

**<h2>O que é o Gov Hub BR?</h2>**

O **Gov Hub BR** é uma plataforma de dados públicos desenvolvida pelo **Lab Livre da UnB** com o apoio do **IPEA**. Seu principal objetivo é transformar dados brutos de sistemas estruturantes do governo federal em informações estratégicas, visuais e acessíveis para gestores públicos e para a sociedade.

**Stack Tecnológica:**

| Ferramenta | Função no Projeto |
| :---: | :---: |
| **Apache Airflow** | Orquestração de pipelines de dados |
| **dbt** | Transformação e modelagem de dados |
| **Apache Superset** | Criação de visualizações e dashboards |
| **PostgreSQL** | Banco de dados relacional |
| **Docker** | Containerização e ambiente |

---

**<h2>Navegando pela Documentação</h2>**

Utilize o menu lateral para explorar as seções do nosso trabalho:
