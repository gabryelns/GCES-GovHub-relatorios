# Contribuições GCES ao GovHub 🌿

Bem-vinda! Este espaço documenta as contribuições realizadas ao projeto **Gov Hub BR** no contexto da disciplina de **Gerência de Configuração e Evolução de Software (GCES)** da Universidade de Brasília.

---

## O que é o Gov Hub?

O **Gov Hub BR** é uma plataforma open source desenvolvida pelo Lab Livre da UnB com apoio do IPEA. Seu objetivo é transformar dados públicos de sistemas estruturantes do governo federal em informações estratégicas para gestores públicos.

| Ferramenta | Função |
|---|---|
| Apache Airflow | Orquestração de pipelines |
| dbt | Transformação e modelagem de dados |
| Apache Superset | Visualização e dashboards |
| PostgreSQL | Banco de dados relacional |
| Docker | Containerização |
